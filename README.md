# True-RNG-random-number-generator
This project shows a TRNG (True Random Number Generator) based on transistor avalance noise. Unique in this project is the integration of a chi-square test into the microcontroller firmware to check for a uniform distribution. It is based on an Arduino Nano V3.

You find a more detailed explanation (in German language) here: https://art-of-electronics.blog/messtechnik/#trueRNG
